**www.openbiometrics.org**

    $ git clone https://github.com/biometrics/openbr.git
    $ cd openbr
    $ git submodule init
    $ git submodule update
    
To optionally check out a particular [tagged release](https://github.com/biometrics/openbr/releases):

    $ git checkout <tag>
    $ git submodule update
